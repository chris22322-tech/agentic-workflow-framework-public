# Templates

Blank, copy-paste-ready versions of each stage's output artifact. Fill them in as you work through the [methodology stages](../stages/index.md).

These templates match the structure used in the [worked example](../worked-example/index.md).

---

| Template | Stage | Output Artifact | Spreadsheet |
|---|---|---|---|
| [Workflow Inventory](workflow-inventory.md) | [1. Decompose](../stages/01-decompose.md) | Workflow Inventory Table | [:material-download:](../downloads/workflow-inventory.xlsx) |
| [Selection Record](selection-record.md) | [2. Select](../stages/02-select.md) | Selection Decision Record | [:material-download:](../downloads/selection-scoring.xlsx) |
| [Scope Document](scope-document.md) | [3. Scope](../stages/03-scope.md) | Workflow Scope Document | [:material-download:](../downloads/scope-document.xlsx) |
| [Design Document](design-document.md) | [4. Design](../stages/04-design.md) | Design Document | — |
| [Evaluation Report](evaluation-report.md) | [6. Evaluate](../stages/06-evaluate.md) | Evaluation Report | — |

!!! note
    There is no template for Stage 5 (Build). The output of that stage is working code, which depends entirely on your chosen workflow, data sources, and environment. See the [Build stage](../stages/05-build.md) for project structure conventions and implementation patterns.
